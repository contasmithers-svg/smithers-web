## Imágenes
Esta versión usa imágenes públicas alojadas en el CDN actual de Wix de Smithers. Así la web se ve visualmente desde el primer minuto sin tener que descargar material.

## Antes de publicar
1. Sustituir/confirmar URL real de Glovo.
2. Completar aviso legal.
3. Confirmar horarios exactos.
4. Confirmar si el enlace de reservas será teléfono, Wix Bookings, CoverManager, TheFork o Google Reserve.
5. Reemplazar con fotos originales si se dispone de ellas.

## Subida
Subir todo el contenido de la carpeta a la raíz del hosting o a Netlify/Vercel/Cloudflare Pages.
